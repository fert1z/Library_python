# Library Python

## Описание
Простой прототип системы управления библиотекой с моделями книг, читателей и выдач. Проект будет развиваться на Python и служит учебным примером.

## Требования
- Python 3.x

## Быстрый старт
1. Создайте и активируйте виртуальное окружение: `py -3 -m venv venv`.
2. Установите зависимости из `requirements.txt` (пока файл пустой).
3. Запустите приложение: `python main.py`.
